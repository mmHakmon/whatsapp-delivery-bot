## 🎯 מקרי שימוש מתקדמים

### 1. שליח רוצה לדעת כמה הרוויח השבוע
```
→ נכנס לדשבורד
→ רואה "רווחים השבוע: ₪565.00"
→ לוחץ על "היסטוריה"
→ בוחר "השבוע"
→ רואה את כל 12 המשלוחים עם הסכומים
```

### 2. לקוח רוצה לדעת איפה המשלוח
```
→ פותח את לינק המעקב
→ רואה סטטוס "נאסף"
→ רואה את טיימליין:
  ✅ נוצר
  ✅ פורסם
  ✅ נתפס
  ✅ נאסף
  ⏳ נמסר (בדרך)
→ יכול להתקשר לשליח
```

### 3. מנהל רוצה לנתח ביצועים
```
→ נכנס לדשבורד מנהלים
→ בוחר "30 ימים"
→ רואה גרפים של:
  - הכנסות יומיות
  - משלוחים יומיים
  - התפלגות שעות
→ רואה שליחים מובילים
→ מייצא נתונים לאקסל לניתוח מעמיק
```
